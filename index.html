Privacy Policy – Pastel Chat

Pastel Chat is a simple private messaging application for friends.

Information We Collect

  • Anonymous user identifier (Firebase UID)
  • Messages sent within the app
  • Message timestamps

No personal information such as name, email address, phone number, or location is collected.

How We Use Information

Stored data is used only to provide chat functionality inside the app.

We do not sell data, share data with third parties, use data for advertising, or track users across apps or websites.

Data Storage

All data is stored securely using Firebase services.
